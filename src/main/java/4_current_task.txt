Урок номер 4.

Цели урока:
    1. Научиться получать данные со страницы и использовать их в коде.
    2. Попробовать поработать с коллекциями элементов со страницы
    3. Написать тест "Добавление товара в корзину" (тест-кейс указан далее)

        Тест-кейс:
            1. Авторизоваться
            2. Выбрать из списка "Sauce Labs Bike Light"
            3. Нажать на него (перейти в карточку товара)
            4. Добавить его в корзину
            5. Перейти в корзину
            6. Товар соответствует ожидаемому ("Sauce Labs Bike Light")

    Не обязательно, но будет большим плюсом:
        1. Вынести авторизацию в отдельный метод, т.к. она будет нужна в большинстве кейсов и есть смысл в
            том, чтобы вынести реализацию. Это позволит избавиться от повторяемости кода относительно проекта
            и углубиться в понимание работы методов.
